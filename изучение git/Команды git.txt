доступные команды с кратким описанием:
git init - инициализировать папку  в которой я нахожусь и создать в папке файл
 git
git config -- local(global) user.name "username" - создать учетную запись с
которой будет вестись работа, локально или глобально
git config --local(global) user.email email - указать почтовый адрес учетной
записи
git pull - получить обновленный файл с серверного репозитория
git status - проверить статус репозитория
git add -A - добавить все файлы в репозитории в отслеживание
git add example.example - добавить конркетный файл по названию в отслежива
ние
git commit -a -m"exmple" - "закоммитить" файлы
git log - посмотреть лог коммитов
git push - "запушить" файлы в онлайн репозиторий
/команды не из гита
cd .. - выйти из папки
cd example - зайти в папку
/
git clone exampleurl examplename - скачать репозиторий из серверного
хранилища на компьютер
